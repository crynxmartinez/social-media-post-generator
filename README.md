# Social Media Post Generator

A local web app to create carousel posts for LinkedIn and Facebook.

## Features

- **PDF Carousel (LinkedIn)** - Generate swipeable PDF carousels
- **Image ZIP (Facebook)** - Export as multiple PNG images
- **6 Themes** - Clean White, Dark Mode, Soft Cream, Ocean Blue, Mint Fresh, Sunset
- **Profile Persistence** - Your name, handle, photo, and theme are saved locally
- **ChatGPT Prompt** - Included prompt template to generate quotes

## Usage

1. Upload your profile photo
2. Enter your name and handle
3. Paste quotes (one per line, first line = title)
4. Pick a theme
5. Download as PDF (LinkedIn) or Images ZIP (Facebook)

## Run Locally

```bash
npm install
npm run dev
```

## ChatGPT Prompt

See `chatgpt-prompt.txt` for a ready-to-use prompt to generate carousel content.
